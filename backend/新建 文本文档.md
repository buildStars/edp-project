# EDP企业培训管理系统 - 功能说明手册

> **北京大学汇丰商学院企业高层管理教育(EDP)项目**  
> 📅 最后更新：2025年11月

---

## 📱 一、小程序端（学员使用）

### 1. 首页

**功能描述**：系统的主入口，展示平台的核心内容和快捷入口。

**主要功能点**：
- **轮播图**：展示重要通知、活动推广、精品课程等
- **课程推荐**：智能推荐适合学员的课程
- **资讯动态**：最新的行业资讯、学习资料
- **协会活动**：校友会、俱乐部活动预告
- **快速搜索**：可搜索课程、资讯、活动

**使用场景**：
- 学员打开小程序后的第一个页面
- 快速了解平台最新动态
- 发现感兴趣的课程和活动

---

### 2. 课程中心

#### 2.1 课程列表页

**功能描述**：浏览和筛选所有可报名的课程。

**主要功能点**：
- **分类筛选**：按课程类型（加速课堂、大师课堂、赋能课堂）筛选
- **状态筛选**：报名中、即将开始、已结束
- **排序方式**：按时间、学分、热度排序
- **课程卡片**：显示课程封面、标题、讲师、时间、地点、学分

**使用场景**：
- 浏览可报名的课程
- 根据兴趣和时间安排选择课程
- 查看课程的基本信息

#### 2.2 课程详情页

**功能描述**：查看课程的完整信息，并进行报名、签到、评价等操作。

**主要功能点**：
- **课程信息**：
  - 课程封面、标题、分类标签
  - 讲师信息（姓名、头像、职称、简介）
  - 开课时间、上课地点
  - 需要的学分（平时课1学分，大师课2学分）
  - 课程介绍（详细的课程内容说明）

- **章节列表**：
  - 显示课程的所有章节
  - 每个章节的标题、排序
  - 章节的学习状态（已签到、已评价）

- **操作按钮**：
  - **立即报名**：使用学分报名课程
  - **申请试听**：如果学分不足，可申请免费试听
  - **申请退课**：已报名但未开始的课程可申请退课
  - **章节签到**：上课时输入签到码完成签到
  - **赠送课程**：已报名的课程可以生成礼物码赠送给好友
  - **查看结课海报**：课程完成后可查看精美的结课海报

**使用场景**：
- 决定是否报名某门课程
- 查看课程的详细内容和讲师介绍
- 上课时进行签到
- 课程结束后查看完成成果

#### 2.3 章节详情页

**功能描述**：查看单个章节的详细内容，完成签到、评价、查看课件的流程。

**主要功能点**：
- **章节信息**：章节标题、开始时间、结束时间
- **学习流程**（按顺序解锁）：
  1. **签到**：输入老师提供的6位签到码完成签到
  2. **评价**：签到后才能评价（评分、评价内容、是否匿名）
  3. **课件资料**：评价后才能查看和下载课件

- **签到记录**：可查看该章节的历史签到记录

**使用场景**：
- 上课时签到打卡
- 课后对课程进行评价反馈
- 下载课程相关的学习资料

**流程说明**：
```
开始上课 → 签到打卡 → 课后评价 → 下载课件
   ↓           ↓           ↓           ↓
 输入签到码   评分评语    查看解锁    PDF/文档
```

### 3. 我的（个人中心）

#### 3.1 个人信息区

**功能描述**：展示用户的基本信息和身份标识。

**显示内容**：
- **头像和昵称**：可点击编辑修改
- **企业身份标识**：
  - 企业管理员：显示"企业管理员"徽章
  - 企业用户：显示"企业用户"徽章
  - 企业名称：所属企业的名称
- **学分信息**（三种学分）：
  - **总学分**：所有学分的总和
  - **个人学分**：可用于赠送课程的学分
  - **锁定学分**：企业管理员分配的学分，只能用于购买课程，不能赠送

**学分说明**：
- 个人学分来源：完成课程奖励、管理员发放、他人赠送
- 锁定学分来源：企业管理员分配给员工
- 使用优先级：报名课程时优先使用锁定学分

#### 3.2 功能菜单（4个快捷入口）

**1. 我的课程**
- **功能**：查看已报名的所有课程
- **分类**：
  - 进行中：正在学习的课程
  - 已完成：已经结课的课程
- **操作**：点击进入课程详情，可继续学习

**2. 我的收藏**
- **功能**：查看收藏的课程、资讯、活动
- **操作**：快速访问感兴趣的内容

**3. 消息中心**
- **功能**：接收系统通知和消息
- **消息类型**：
  - 报名成功通知
  - 开课提醒
  - 签到提醒
  - 学分变动通知
  - 系统公告
- **提示**：未读消息会显示红点或数字角标

**4. 课件下载**
- **功能**：查看已下载的所有课件资料
- **内容**：PDF文档、课程讲义、学习资料
- **操作**：可重新打开查看或分享

#### 3.3 企业功能区

**仅企业管理员和企业用户可见**

**企业管理员功能**：
- **员工管理**：
  - 查看企业下所有员工列表
  - 查看员工的学分余额（个人学分、锁定学分）
  - 为员工分配锁定学分
  - 为员工购买课程
- **学分分配规则**：
  - 使用自己的个人学分分配给员工
  - 分配给员工的学分会变成锁定学分
  - 员工只能用这些学分购买课程，不能赠送

**企业用户功能**：
- **我的企业**：
  - 查看所属企业名称
  - 查看企业管理员信息
  - 查看自己的学分情况

#### 3.4 其他功能入口

**签到记录**
- **功能**：查看自己的所有签到历史
- **信息展示**：
  - 签到统计：总次数、准时次数、迟到次数
  - 签到详情：每次签到的课程、章节、时间、状态

**AI学习报告**
- **功能**：AI分析生成个人学习报告
- **内容**：学习情况分析、课程推荐、学习建议

**课程赠送**
- **功能**：将自己报名的课程赠送给好友
- **流程**：
  1. 选择要赠送的课程
  2. 生成礼物码
  3. 分享给好友
  4. 好友使用礼物码领取课程

**报名/退课申请**
- **报名申请列表**：查看试听申请的审批状态
- **退课申请列表**：查看退课申请的处理进度

**编辑资料**
- **可修改内容**：
  - 头像（点击头像可选择微信头像或相册图片）
  - 昵称
  - 公司名称
  - 职位
  - 手机号（点击授权绑定微信手机号）

### 4. 资讯中心

**功能描述**：浏览行业资讯、学习资料、政策解读等内容。

**主要功能点**：
- **资讯列表**：按时间倒序展示
- **资讯分类**：行业动态、政策法规、学习资料
- **资讯详情**：
  - 标题、封面图
  - 发布时间、浏览量
  - 富文本内容（支持图文混排）

- **搜索功能**：按关键词搜索资讯

### 5. 协会中心

**功能描述**：校友会、俱乐部等协会组织的展示和活动管理。

**主要功能点**：
- **协会列表**：展示所有协会
- **协会详情**：
  - 协会简介、封面图
  - 成员数量、活动数量
  - 协会最新动态
- **活动列表**：该协会举办的活动
- **活动详情**：
  - 活动信息（时间、地点、人数限制）
  - 活动介绍
  - 报名按钮
  - 活动相册

---

### 6. 学分中心

**功能描述**：查看学分明细和使用记录。

**主要功能点**：
- **学分总览**：
  - 总学分余额
  - 个人学分（可赠课）
  - 锁定学分（只能购课）
  - 累计获得学分
  - 累计消耗学分

- **学分记录**：
  - 记录类型：获得、消耗、退回、分配、赠送
  - 每条记录显示：
    - 时间
    - 类型
    - 数量（+或-）
    - 关联课程
    - 备注说明
    - 余额变化

- **学分获取方式**：
  - 完成课程学习（系统自动发放）
  - 管理员手动发放
  - 企业管理员分配（变成锁定学分）
  - 接受他人赠送的课程

- **学分使用方式**：
  - 报名课程（扣除学分）
  - 赠送课程给他人（使用个人学分）

**主要功能点**：
- **用户列表**：
  - 显示：头像、昵称、手机号、角色、学分、注册时间
  - 筛选：按角色、状态筛选
  - 搜索：按昵称、手机号搜索

- **批量操作**：
  - 批量导入用户
  - 批量发放学分
  - 批量发送通知

---

### 3. 课程管理

#### 3.1 课程列表

**功能描述**：管理所有课程。

**主要功能点**：
- **课程列表**：
  - 显示：封面、标题、分类、学分、报名人数、状态
  - 筛选：按分类、状态筛选
  - 搜索：按课程名称搜索

- **课程状态**：
  - 草稿：未发布
  - 待审核：等待审批
  - 已发布：可以报名
  - 进行中：正在上课
  - 已结束：课程结束

- **操作**：
  - 新建课程
  - 编辑课程
  - 删除课程
  - 发布/下架
  - 查看报名学员

#### 3.2 创建/编辑课程

**功能描述**：创建新课程或编辑现有课程。

**课程信息配置**：
1. **基本信息**：
   - 课程标题
   - 课程封面图
   - 课程分类（加速/大师/赋能课堂）
   - 所需学分（平时课1分，大师课2分）

2. **课程详情**：
   - 课程介绍（富文本编辑）
   - 开始时间、结束时间
   - 上课地点
   - 最大报名人数

3. **讲师信息**：
   - 选择讲师（从教师用户中选择）
   - 讲师姓名
   - 讲师头像
   - 讲师职称
   - 讲师简介

4. **章节管理**：
   - 添加章节
   - 章节标题
   - 章节排序
   - 章节状态（草稿/已发布）

5. **结课设置**：
   - 要求签到次数：学员需要签到多少次才能结课
   - 结课学分奖励：完成课程后奖励的学分数

6. **报名设置**：
   - 报名状态：开放报名/关闭报名/需审核
   - 报名开始时间
   - 报名截止时间

#### 3.3 章节管理

**功能描述**：管理课程的章节内容。

**主要功能点**：
- **章节列表**：显示课程下的所有章节
- **章节编辑**：
  - 章节标题
  - 章节介绍
  - 上课时间
  - 排序调整
- **上传课件**：为章节上传PDF、文档等学习资料

#### 3.4 课程学员管理（教师功能）

**功能描述**：教师查看和管理自己课程的学员。

**主要功能点**：
- **学员列表**：
  - 显示：头像、昵称、手机号、签到次数、课程状态
  - 签到统计：应到X次，实到X次
  - 课程状态：进行中、已完成

- **单个操作**：
  - **发放海报**：手动给单个学员发放结课海报，标记为已完成
  - 查看学员详情

- **批量操作**：
  - **批量发放海报**：
    - 选择多个学员
    - 一键发放结课海报
    - 系统自动标记为已完成

- **筛选功能**：
  - 按完成状态筛选（进行中/已完成）
  - 按签到次数筛选（达标/未达标）
  - 只显示签到达标的学员

**结课流程说明**：
1. 教师设置课程的"要求签到次数"（如：需签到5次）
2. 学员上课时签到打卡
3. 课程结束后，教师有两种方式让学员结课：
   - **方式一（批量）**：选择签到达标的学员，批量发放结课海报
   - **方式二（单个）**：给特定学员单独发放结课海报（不管是否达标）
4. 学员标记为"已完成"后，会收到精美的结课海报
5. 学员获得结课学分奖励

## 💡 四、关键概念说明

### 学分系统

**学分的作用**：
- 学分是平台的虚拟货币
- 用于报名课程（不同课程需要不同学分）
- 完成课程后可获得学分奖励

**学分类型**：
1. **个人学分**：
   - 完全由学员自己控制
   - 可以用于报名课程
   - 可以赠送课程给他人
   
2. **锁定学分**：
   - 由企业管理员分配
   - 只能用于报名课程
   - 不能赠送给他人
   - 使用时优先扣除

**学分流转**：
```
获得学分：
- 完成课程 → +学分（系统自动）
- 管理员发放 → +学分（手动）
- 企业分配 → +锁定学分

消耗学分：
- 报名课程 → -学分（优先用锁定学分）
- 赠送课程 → -个人学分（只能用个人学分）

退回学分：
- 退课成功 → +学分（退回个人学分）
```

### 课程分类

1. **加速课堂**：
   - 面向中层管理者
   - 1学分/课程
   - 快速提升管理技能

2. **大师课堂**：
   - 邀请行业大师授课
   - 2学分/课程
   - 高端、深度的课程内容

3. **赋能课堂**：
   - 实战技能培训
   - 1学分/课程
   - 注重实操和应用

### 签到系统

**签到的意义**：
- 确保学员真实参加课程
- 作为结课的依据（需要达到要求的签到次数）
- 解锁评价和课件下载功能

**签到方式**：
- 教师创建签到会话，生成6位数字签到码
- 学员在有效时间内（如30分钟）输入签到码
- 系统记录签到时间和状态（准时/迟到）

**签到流程**：
```
教师端：创建签到会话 → 生成签到码 → 告知学员
学员端：打开课程详情 → 点击签到 → 输入签到码 → 完成签到
```

### 结课机制

**结课条件**：
- 课程必须已结束
- 学员签到次数达到要求（由教师设置）
- 教师手动发放结课海报

**结课方式**：
1. **批量结课**：
   - 教师筛选出签到达标的学员
   - 批量发放结课海报
   - 系统自动标记为"已完成"并发放学分

2. **单独结课**：
   - 教师给特定学员单独发放海报
   - 不管该学员是否签到达标
   - 用于特殊情况处理

**结课后的效果**：
- 学员状态变为"已完成"
- 学员获得精美的结课海报
- 学员获得学分奖励
- 首次查看时自动弹出海报
- 可以随时在课程详情查看海报
- 可以分享海报到朋友圈

## 📱 六、常见使用场景

### 场景1：新员工入职培训

**情况**：公司新入职10名员工，需要统一培训

**操作流程**：
1. 管理员在后台创建企业账户
2. 管理员将10名员工添加为企业用户
3. 企业管理员给每名员工分配20个锁定学分
4. 企业管理员选择2门必修课程（共4学分）
5. 统一为10名员工报名这2门课程
6. 员工在小程序接收通知，按时上课
7. 课程结束后，企业管理员查看学习完成情况
8. 生成企业培训报告

### 场景2：学员学分不足想听课

**情况**：学员学分不足，但非常想参加某门大师课

**操作流程**：
1. 学员在课程详情页发现需要2学分，但自己只有1学分
2. 点击"申请试听"按钮
3. 填写申请理由（如：对课程内容非常感兴趣，希望免费试听）
4. 提交申请
5. 管理员在后台审批
6. 审批通过后，学员免费参加课程
7. 学员正常签到、评价、学习
8. 课程结束后仍可获得学分奖励

### 场景3：学员推荐课程给朋友

**情况**：学员觉得课程很好，想推荐给朋友一起学习

**操作流程**：
1. 学员在课程详情页点击"赠送课程"
2. 系统生成唯一的礼物码（如：ABC123）
3. 学员通过微信分享给朋友
4. 朋友打开小程序，自动进入领取页面
5. 朋友点击"领取课程"
6. 系统自动为朋友报名（不扣学分）
7. 两人可以一起学习讨论

### 场景4：学员临时有事无法参加

**情况**：学员已报名课程，但突然有急事无法参加

**操作流程**：
1. 学员在"我的课程"找到已报名的课程
2. 进入课程详情页
3. 点击"申请退课"
4. 填写退课理由（如：出差无法参加）
5. 提交退课申请
6. 管理员审批
7. 同意后，学分退回到学员账户
8. 学员可以用学分报名其他课程

### 场景5：教师课程结束发放证书

**情况**：课程全部结束，需要给学员发放结课证明

**操作流程**：
1. 教师登录管理后台
2. 进入"课程学员管理"
3. 查看学员签到统计（如：要求5次，已签到5次）
4. 筛选出签到达标的学员（共15人）
5. 批量勾选这15人
6. 点击"批量发放海报"
7. 系统自动：
   - 生成15份精美结课海报
   - 标记这15人为"已完成"
   - 发放学分奖励（如每人+2学分）
   - 发送通知给学员
8. 学员打开小程序，看到弹出的结课海报
9. 学员可以保存海报或分享到朋友圈
