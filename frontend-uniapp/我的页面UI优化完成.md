## 🎯 设计原则

### 1. 色彩系统
- **主色：** #C8161D (北大汇丰红)
- **功能色：**
  - 蓝色：#1890FF (课程、信息)
  - 紫色：#722ED1 (消息、通知)
  - 绿色：#52C41A (下载、完成)
  - 橙色：#FF6B00 (客服、警告)
  - 金色：#FFD700 (学分、奖励)

### 2. 渐变规范
- **红色渐变：** `linear-gradient(135deg, #C8161D 0%, #A0141A 100%)`
- **蓝色渐变：** `linear-gradient(135deg, #1890FF 0%, #096DD9 100%)`
- **紫色渐变：** `linear-gradient(135deg, #722ED1 0%, #531DAB 100%)`
- **绿色渐变：** `linear-gradient(135deg, #52C41A 0%, #389E0D 100%)`

### 3. 圆角规范
- **大卡片：** 24rpx
- **中等卡片：** 20rpx
- **小卡片：** 16rpx
- **按钮：** 48rpx (胶囊形)
- **圆形：** 50%

### 4. 间距规范
- **页面边距：** 24-32rpx
- **卡片内边距：** 28-32rpx
- **元素间距：** 20-24rpx
- **小间距：** 12-16rpx

### 5. 阴影规范
- **卡片阴影：** `0 4rpx 20rpx rgba(0, 0, 0, 0.04)`
- **悬浮阴影：** `0 6rpx 24rpx rgba(0, 0, 0, 0.08)`
- **强调阴影：** `0 8rpx 24rpx rgba(0, 0, 0, 0.1)`
- **彩色阴影：** 根据按钮颜色，透明度0.3-0.4

### 6. 特效使用
- **毛玻璃效果：** `backdrop-filter: blur(20rpx)`
- **发光效果：** 半透明背景 + blur滤镜
- **装饰圆圈：** 半透明圆形元素点缀背景

## 📱 交互优化

### 1. 点击反馈
- **卡片：** `transform: scale(0.95)` + 阴影加深
- **按钮：** `transform: scale(0.98)` + 背景色变化
- **图标：** 缩放 + 颜色渐变

### 2. 过渡动画
- **标准时长：** 0.3s
- **缓动函数：** `cubic-bezier(0.4, 0, 0.2, 1)`
- **滑动指示器：** 平滑移动过渡

### 3. 状态反馈
- **激活状态：** 颜色变化 + 字重加粗
- **禁用状态：** 降低透明度
- **加载状态：** 文字提示

## 🚀 性能优化

### 1. 渲染优化
- 使用CSS动画代替JS动画
- transform启用GPU加速
- 合理使用backdrop-filter（注意兼容性）

### 2. 布局优化
- Grid布局提升渲染效率
- Flex布局灵活适配
- 固定高度减少重排

### 3. 资源优化
- 使用Icon组件统一管理图标
- 渐变色替代图片背景
- SVG图标体积小、清晰度高
