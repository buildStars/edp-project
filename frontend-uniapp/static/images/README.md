## 图片规格说明

1. **文件格式**：建议使用PNG格式（支持透明背景）
2. **图标尺寸**：
   - 小图标（如箭头）：32x32px
   - 中等图标（如功能图标）：48x48px
   - Tab栏图标：81x81px
   - 头像：200x200px
   - 空状态图：400x300px

3. **颜色规范**：
   - 主题色：#C8161D（北大红）
   - 灰色图标：#999999
   - 选中/激活状态：#C8161D

4. **注意事项**：
   - 所有图标应提供@2x和@3x版本以适配不同屏幕
   - Tab栏图标需要提供选中和未选中两种状态
   - 确保图标风格统一，符合整体设计规范

## 占位图片

在开发阶段，可以使用以下在线图片生成服务创建占位图：
- https://placeholder.com/
- https://via.placeholder.com/

示例：
- Logo占位：`https://via.placeholder.com/200x60/C8161D/FFFFFF?text=EDP+LOGO`
- 头像占位：`https://via.placeholder.com/200x200/CCCCCC/FFFFFF?text=Avatar`
- 空状态占位：`https://via.placeholder.com/400x300/F5F5F5/999999?text=No+Data`
